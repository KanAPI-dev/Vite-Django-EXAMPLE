# ReactDjango Frontend

ReactDjango's client side

## Features

- Built with [Vite](https://vite.dev/)
- Routing with [Tanstack Router](https://tanstack.com/router/latest)
- Styled with [tailwindcss](https://tailwindcss.com/)

## How to start

1. Make sure [Node.js](https://nodejs.org/en) is installed
2. Make sure [pnpm](https://pnpm.io/installation) is installed
3. `cd` frontend
4. Optional: `pnpm install` (only if there are new/updated dependencies)
5. `pnpm dev`
